# Nam Viet Group — Corporate Website

Bản **chapter storytelling** (HTML / CSS / JS thuần): tiêu đề lớn Fraunces, số chương, copy song ngữ EN/VN, animation scroll.

## Chạy

```bash
python -m http.server 8123
# http://localhost:8123
```

## Cấu trúc

- `index.html` — 12 chapter
- `css/style.css` — design system editorial
- `js/main.js` — EN/VI, reveal, counters, parallax, chapter rail
- `assets/img/` — ảnh chủ đạo
